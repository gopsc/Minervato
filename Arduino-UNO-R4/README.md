## COPS_ARDUINO

这是部署在ARDUINO UNO R4之类板子上面的外部模块，板子需要安装W5500 ETH 以太网扩展板。

## 程序的作用

这个程序会启动一个HTTP协处理器服务，能够根据收到的请求访问传感器模块，并且传回响应。

## 使用方法
```bash
curl -X POST http://192.168.254.101:8001/api -d '{"cmd": "MPU6050"}'
```


## 启动流程

1. 板载基础模块首先进行启动
2. 项目启动，探测传感器
3. 启动显示模块，成功则进行日志分级记录
4. 启动网络模块，获取IP地址，访问网关（失败则自己做服务器）获取传感器上报阈值。
5. 如果没有网络，就进行持续输出（屏幕或串口）
